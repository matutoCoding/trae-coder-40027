# 危化品罐区储运业务管理系统

一款用于危化品储运公司管理收发、储罐和安全的客户端软件。

## 功能模块

### 1. 储罐台账
- 储罐基础信息管理（编号、名称、类型、介质、容量、材质等）
- 储罐状态监控（液位、体积、温度、压力）
- 储罐增删改查操作
- 储罐详情查看

### 2. 收料计量
- 进料计量记录管理
- 供应商、车辆信息登记
- 收前收后体积/液位对比
- 温度、密度计量
- 收料记录查询筛选

### 3. 发料装车
- 罐车装卸记录管理
- 客户单位信息管理
- 发前发后体积/液位对比
- 发料记录查询筛选
- 装车作业管理

### 4. 罐区巡检
- 罐区定点巡检记录
- 检查项目管理
- 异常问题记录
- 防雷防静电检查
- 巡检人员管理

### 5. 氮封惰化
- 氮封压力实时监控
- 氮封压力控制记录
- 阀门状态管理
- 压力设定值管理

### 6. 消防联动
- 消防泡沫联动管理
- 消防设备台账
- 围堰积液排放管理
- 应急预案演练
- 演练记录与总结

### 7. 安全监测
- 液位温度实时监控
- 可燃/有毒气体检测
- 气体探测器状态监控
- 动火作业许可管理
- 作业审批流程

## 技术栈

- **语言**: Python 3.8+
- **GUI框架**: PySide6 (Qt for Python)
- **架构**: 模块化设计

## 安装运行

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 运行程序

```bash
python main.py
```
